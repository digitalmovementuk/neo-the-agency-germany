# NEO THE AGENCY Germany

German market entry site for NEO THE AGENCY, built with Vite, React, TypeScript, Tailwind CSS, and React Router.

## Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run test:e2e`
